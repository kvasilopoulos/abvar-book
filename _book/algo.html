8.1.2 Algorithm 1

  1. Create a 2D-array of \(Kp \times (h+1) \times Kp\)
  2. Assign the shock to the first observation (unit, se, etc)
  3. Iterate over the horizons 1, 2, …, h where \[ irf_{i+1} = irf_{i} A'\]

8.1.3 Algorithm 2

  1. Create a array \(J = [I_k, 0_{K\times K(p-1)}]\)
  2. Create an 3D-array of \(Kp \times (h + 1) \times Kp\) to store the irfs
  3. Assign the shock to the first observation (unit, se, etc)
  4. Iterate over the horizons for i = 1,2, …, h where \[ irf_{i+1} = J A^i J'\]
